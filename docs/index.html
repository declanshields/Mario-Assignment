Mario Project

Project Explanation

This page is dedicated to the Mario project created for Games Engine Creation at Staffordshire University.

Demonstration of the project.

If the video above doesn't work, it can be found in the project folder on GitHub

This project was created during semester 2 of Games Engine Creation and is what I submitted to be marked. It is a project created in the C++ language using the SDL framework to handle sprite rendering and sound. This project is a simple immitation of a Mario game on a smaller scale using sprites that were provided during the lecture. This project includes collision between the level map and the two player characters, as well as gravity and sprite collision between the player characters and the enemies.
